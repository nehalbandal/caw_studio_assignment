Ability to check all cinemas in which a movie is playing along with all the showtimes

Ability to book a ticket. (No payment gateway integration is required. Assume tickets can be booked for free)

The API to book a ticket should be protected i.e. only a logged-in user should be allowed to access that API. Rest, all other APIs are public endpoints
